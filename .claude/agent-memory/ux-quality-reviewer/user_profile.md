Разработчик и архитектор PyQt6-приложения GeoVertical Analyzer — специализированного ПО для геодезического контроля антенно-мачтовых сооружений (АМС). Работает с пользователями-геодезистами, знакомыми с российскими стандартами (СП 70.13330.2012, ГОСТ Р 71949-2025).

Запрашивал полный UX-аудит (2026-03-28) по всем 9 GUI-компонентам. Аудит охватывает информационную архитектуру, обратную связь, обработку ошибок, консистентность, рабочие процессы, доступность.
